# 展开运算符

展开运算符在处理数组时会去重。如果在展开过程中出现重复的元素，JavaScript 会自动去除重复的部分，确保最终结果中不含重复元素。

例如：

```javascript
const arr1 = [1, 2, 3];
const arr2 = [3, 4, 5];
const arr3 = [...arr1, ...arr2];
console.log(arr3); // 输出：[1, 2, 3, 4, 5]
```

在上面的例子中，当将 arr1 和 arr2 展开后合并为 arr3 时，重复的元素 3 只会出现一次，确保了最终的合并数组不含重复元素。
