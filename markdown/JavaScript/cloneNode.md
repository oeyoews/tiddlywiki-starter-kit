# cloneNode

cloneNode 是一个 DOM API 中的方法，用于复制一个节点并返回它的副本。该方法可以接受一个布尔值参数 deep，表示是否复制节点的子节点。如果 deep 为 true，则会同时复制该节点的所有子孙节点，否则只复制该节点本身。
