# Nodelist hidden

是的，HTML节点有一个hidden属性，它是一个布尔属性。当设置为true时，该属性将隐藏该节点，使其在页面中不可见。要使用hidden属性，只需将其添加到HTML标签中即可，例如：

```html
<div hidden>这是一个隐藏的节点</div>
```

```js
xxxNodelist.hidden = true
```

在上面的示例中，div元素被隐藏起来，因此在页面上不可见。
