# 命令式-声明式

```js
//命令式
let n = [-9, 87, 72, 452, 32, -9];
for (let i = 0; i < n.length; i++) {
  console.log(n[i]);
}

//声明式（函数式，封装性更好）
let n = [-9, 87, 72, 452, 32, -9];
n.forEach((v) => console.log(v));
```
