# 不可变数据02

```js
class MyClass {
  constructor(x) {
    // Use a different name, such as `_x`, to store the property value.
    this._x = x === undefined ? 0 : x;
  }

  // Use a different name for the getter method.
  get x() {
    return this._x;
  }
}

const c = new MyClass();
c.x = 99;
console.log(c.x); // 0
```

